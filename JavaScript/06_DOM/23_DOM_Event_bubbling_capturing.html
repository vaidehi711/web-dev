Event Bubbling: Jab koi event ho to wo pehle inner element pe chala jata hai phir uske parent element pe chala jata hai.
    Event Capturing: Jab koi event ho to wo pehle parent element pe chala jata hai phir uske inner element pe chala jata hai.
    Agar aap "Event bubbling" ko rokna chahte hain to aap event.stopPropagation() method ka use kar sakte hain.
