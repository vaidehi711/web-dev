JavaScript Kya Hai?

JavaScript ek programming language hai jo web pages ko interactive banane mein help karti hai. Isse aap dynamic content create kar sakte ho, images ko animate kar sakte ho, multimedia control kar sakte ho aur bahut kuch kar sakte ho. JavaScript high-level interpreted language hai jo ECMAScript standard ko follow karti hai. Iska syntax curly braces ka use karta hai, dynamic typing hoti hai, aur object-oriented concepts ko support karta hai.

JavaScript Kya-Kya Kar Sakta Hai?

JavaScript se aap yeh sab kar sakte ho:

  • HTML content ko change karna
  • HTML attribute values ko change karna
  • HTML CSS styles ko change karna
  • HTML elements ko hide aur show karna
  • Form data ko validate karna
  • Cookies create karna
  • Visitor ke browser ko detect karna
  • Visitor ki screen size pata karna
  • Mouse position detect karna
  • Keyboard key press detect karna
  • Alert, prompt aur confirmation pop-ups dikhana

JavaScript Syntax (Likhnay ka Tarika)

  • JavaScript case-sensitive hota hai (capital aur small letters alag maane jaate hain)
  • Statements ko semicolon se end kiya jata hai
  • Code blocks curly braces ke andar hote hain { }
  • JavaScript ko HTML file ke andar likha jata hai
  • JavaScript ko external file mein bhi likha ja sakta hai

JavaScript Statements

Statements ke andar yeh sab hota hai:

  • Values (jaise 5, "Hello")
  • Operators (jaise +, -, *, /)
  • Expressions (jaise x + y)
  • Keywords (jaise var, function)
  • Comments (// single line, /* multi-line */)

JavaScript Variables (Data Store Karne ke Liye)

Variables containers hote hain jisme data store hota hai.

        var x = 5;
        var y = 10;
        var z = x + y;  // z ka value hoga 15

JavaScript Operators

Operators ka use calculation aur comparison ke liye hota hai.

        var a = 20;
        var b = 10;
        var sum = a + b;  // Addition
        var diff = a - b; // Subtraction
        var mul = a * b;  // Multiplication
        var div = a / b;  // Division

JavaScript Data Types

Data types matlab kis tarah ka data variable hold kar raha hai:

        var age = 25;                          // Number
        var name = "Meghraj";                 // String
        var student = {name:"Ravi", age:22}; // Object me key-value pairs hoti hain
        var isPass = true;                    // Boolean

JavaScript Functions (Kaam Karwane ke Liye Code ka Block)

Function ek block hota hai jo koi bhi task perform karta hai.

        function multiply(p1, p2) {
            return p1 * p2;
        }
        var result = multiply(4, 5); // Output: 20

JavaScript Objects (Properties aur Methods Wale Containers)

        var person = {
            firstName: "Meghraj",
            lastName: "Hamade",
            age: 28,
            city: "Nanded-26"
        };

JavaScript Events (User Actions ko Detect Karna)

JavaScript mein aap kisi action par code chala sakte ho jaise click, mouseover, etc.

JavaScript Strings (Text Ko Represent Karne ke Liye)

        var car1 = "BMW";   // Double quotes
        var car2 = 'Audi';   // Single quotes

JavaScript String Methods

String methods se text pe operations kar sakte ho:

        var message = "Hello World!";
        var length = message.length;      // Output: 12
        var upper = message.toUpperCase(); // Output: "HELLO WORLD!"
        var lower = message.toLowerCase(); // Output: "hello world!"
        var sliced = message.slice(0, 5); // Output: "Hello"
