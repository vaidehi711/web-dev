Compiler vs Interpretor

Compiler aur Interpretor dono hi programming languages ke code ko machine language mein convert karte hain. Lekin dono mein kuch farak hota hai:

  • Compiler: Compiler ek software hota hai jo pura code ek baar mein hi compile kar deta hai. Iska output ek machine code hota hai jo directly run kar sakta hai.
  • Interpretor: Interpretor ek software hota hai jo code ko line-by-line run karta hai. Iska output ek intermediate code hota hai jo machine code mein convert hota hai aur phir run hota hai.

Compiler ka Example

Compiler ka example hai C, C++, Java, C# etc. In languages ke code ko compile karne ke liye aapko ek compiler ki zarurat hoti hai. Compiler code ko ek baar mein hi pura compile kar deta hai.

Interpretor ka Example

Interpretor ka example hai Python, Ruby, JavaScript etc. In languages ke code ko run karne ke liye aapko ek interpretor ki zarurat hoti hai. Interpretor code ko line-by-line run karta hai.
